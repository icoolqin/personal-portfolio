# OmniWord - 你的 AI 单词说文解字大师

## 不止是“翻译”，而是“理解”
厌倦了只告诉你“什么意思”的翻译工具？OmniWord 不仅仅是词典，更像你的词源学家、文化向导和记忆教练。了解单词的词源、构词和语境，从源头建立牢固记忆。

## 🚀 为什么你会爱上 OmniWord
- AI 说文解字：词源、词根/词缀、文化语境、地道示例，一次看懂。
- 划词即译：网页上选中文本即可呼出翻译，无需跳转。
- 双引擎：日常用有道翻译秒查；需要更深则切换 AI 翻译。
- 你的 AI，你做主：支持 OpenAI / Claude / Gemini / DeepSeek 等，设置里填入你的 API Key 即可。推荐硅基流动 SiliconFlow，免费额度友好。
- 流畅体验：面板可拖动并记忆位置，AI 结果实时流式输出。

## ✨ 核心功能
- 网页划词，即时翻译
- 弹窗查词，快捷检索
- AI 深度讲解（词源/词缀/文化语境）
- 多家 AI 服务商支持（OpenAI、Claude、Gemini 等）
- 支持自定义 API Key
- 可拖动、可记忆位置的翻译窗口

## 🎯 快速开始
1. 安装扩展（Chrome / Edge）
2. 在网页选中单词或短语
3. 点击/悬停 OmniWord 图标查看翻译
4. 设置中填写你的 AI 服务商与 API Key（可选）以开启深度讲解

## ❓ 常见问题
**支持哪些 AI 服务？** OpenAI、Claude、Gemini、DeepSeek 等。自行配置 Key，推荐使用 SiliconFlow 的免费额度。

**会拖慢网页吗？** 扩展轻量，响应流畅；AI 输出为流式展示。

**会收集我的数据吗？** 不会。个人数据与 API Key 仅保存在本地。
